# 🎮 game-art-downloader - Save game images with one click

[![](https://img.shields.io/badge/Download-Latest-blue.svg)](https://github.com/anachronismgenuslolium750/game-art-downloader)

## About the Project ℹ️

The game-art-downloader tool simplifies the process of finding and saving game cover art. Many storefronts make it difficult to save high-quality images directly. This tool acts as a bridge between your browser and these web stores. It recognizes the game page you visit and provides a button to save the artwork to your computer.

You do not need to hunt through page source code. The tool handles the extraction for you. It supports stores from Steam, GOG, Xbox, PlayStation, and Nintendo.

## Prerequisites 🛠️

Your computer needs a web browser to use this tool. Most users rely on Chrome, Firefox, or Microsoft Edge. Ensure your browser remains up to date. You do not need to install complex software or programming environments.

This tool runs as a browser extension or a lightweight script manager. You must install a script manager first. This manager acts as a folder that holds the downloader script.

## Step One: Install a Script Manager 📥

A script manager allows your browser to run small programs on websites.

1. Open your web browser.
2. Search for the "Tampermonkey" extension in your browser store.
3. Click the button to add it to your browser.
4. Confirm the installation if the browser shows a prompt.

The manager icon appears in your toolbar once complete.

## Step Two: Download the Tool ⚙️

Visit this page to download the script: [https://github.com/anachronismgenuslolium750/game-art-downloader](https://github.com/anachronismgenuslolium750/game-art-downloader)

1. Navigate to the link shown above.
2. Locate the link to the latest version of the script.
3. Click the link to begin the install process.
4. Your script manager will detect the file.
5. Click "Install" when your script manager asks for permission to add the new script.

## Using the Downloader 🖱️

Once the installation finishes, you can start saving images.

1. Open a new tab in your web browser.
2. Visit a game store page like Steam or GOG.
3. Refresh the page if the store was already open.
4. Look for a new button labeled "Download Cover" near the game title.
5. Click this button to save the image to your Downloads folder.

The tool automatically identifies the file type and resolution. It fetches the largest version of the art available from the store server.

## Troubleshooting Common Issues 💡

Most issues come from browser settings. Follow these steps if the button does not appear.

*   **Refresh the Page:** Browsers often load store pages before extensions start. Reload the page to force the script to scan for images again.
*   **Check the Manager:** Click the script manager icon in your toolbar. Verify that the "game-art-downloader" script shows a green switch or a checkmark. If it says "Disabled," click the switch to turn it on.
*   **Clear Cache:** Sometimes old data blocks new scripts. Clear your browser cache and restart the application.
*   **Update the Script:** If a specific store changes its website layout, the script might stop working. Visit the download link periodically to check for updates. The script manager usually checks for updates automatically, but a manual reinstall fixes most errors.

## Supported Platforms 🌐

The tool works on these storefronts:

*   **Steam:** Captures header images and capsule art.
*   **GOG:** Finds high-resolution cover art for your library.
*   **Xbox:** Extracts background and game tile imagery.
*   **PlayStation:** Retrieves store listing art.
*   **Nintendo:** Saves eShop game icons and flyers.

## System Performance ⚡

This tool uses minimal memory. It stays dormant until you visit a supported store page. It does not run in the background when you browse other websites. It only consumes resources while you view a game store. You will not notice any slowdowns on your Windows machine while using this tool.

## Privacy and Security 🔒

The script only interacts with the specific web addresses defined in its settings. It does not read your private mail or browser history. It does not send your data to external servers. It only functions to copy the image URL from the game store and trigger a standard browser download action. You maintain control over your files at all times.
